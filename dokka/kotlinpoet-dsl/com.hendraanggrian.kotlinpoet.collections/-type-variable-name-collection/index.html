TypeVariableNameCollection

class TypeVariableNameCollection : MutableCollection<TypeVariableName>

A TypeVariableNameCollection is responsible for managing a set of type variable name instances. Since Kotlinpoet keep TypeVariableName in lists and sets, this class extends Collection.
